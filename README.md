# PamsStore - Responsive Landing Page

## Project Overview
PamsStore is a responsive landing page designed for an online store. The page features a modern and visually appealing layout with a header, product section, testimonials, newsletter signup, promo banner, and footer. Built with **HTML**, **SASS/CSS**, and **Bootstrap 5**, the design ensures optimal display across various screen sizes, from desktop to mobile devices.

---

## Features

### Header / Navbar
- Sticky top navigation for easy access.
- Responsive mobile menu using Bootstrap collapse.
- Navigation links to key sections: Home, Products, Reviews, Newsletter, Contact.
- Brand typography uses **Playfair Display** with accent colors for distinction.

### Hero Section
- Engaging hero banner with headline, subtext, and call-to-action buttons.
- Hero image optimized for responsiveness with subtle decorative overlays.
- Buttons styled to provide clear visual hierarchy: primary "Shop Now" and secondary "Get Deals".

### Features Strip
- Highlights key benefits: Free Shipping, 30-Day Returns, Secure Payments, 24/7 Support.
- Icon-based design for fast visual communication.

### Products Section
- Displays **eight product cards**, each with image, category, title, description, price, and "Shop Now" button.
- Product cards include hover animations for interactivity.
- Badge labels for new, hot, or sale items.
- Fully responsive grid using Bootstrap's row-cols classes and media queries.

### Promo Banner
- Eye-catching gradient banner highlighting seasonal offers.
- CTA button directs users to product section.

### Testimonials
- Displays customer reviews with star ratings.
- Styled cards with subtle shadows to create visual separation.

### Newsletter
- Email subscription form to capture leads.
- Responsive layout ensures easy interaction on mobile devices.

### Footer
- Organized into multiple sections: Brand info, Shop, Company, Support, and Contact.
- Social media icons with hover effects.
- Footer bottom contains copyright and policy links.

## Design Choices

1. **Color Palette**
   - **$color-light** (#ececec) for background to maintain readability.
   - **$color-mint** (#9fd3c7) for accents, buttons, and highlights.
   - **$color-navy** (#385170) and **$color-deep-navy** (#142d4c) for text, navbar, and card borders.
   
2. **Typography**
   - **Playfair Display** for headings to create a premium look.
   - **Lato** for body text for readability and modern feel.

3. **Responsiveness**
   - Bootstrap 5 grid used for flexible layouts.
   - Media queries fine-tune typography, spacing, and card sizes for tablets and phones.
   - Mobile-first approach ensures usability on all devices.

4. **Interactivity**
   - Smooth hover effects on buttons and product cards.
   - Transition effects for subtle animations enhance user experience without distraction.

5. **Code Structure**
   - SASS variables used for colors, fonts, and transitions for maintainability.
   - Semantic HTML5 elements for accessibility and SEO.
   - Comments included in CSS/SASS for clarity.

---

## Technologies Used
- **HTML5** – Structure and semantics
- **SASS/SCSS** – Modular and maintainable styling
- **Bootstrap 5** – Responsive grid and components
- **Google Fonts** – Playfair Display & Lato
- **Bootstrap Icons** – Visual icons for features and social links
